# Netflix

Sistema semelhante ao Netflix, projeto final do curso de Nodejs

# Rodando localmente

    git clone https://github.com/troianosacademy/netflix.git
    cd netflix
    npm install
    npm start

    http://localhost:3000/
    http://localhost:3000/admin

    Default user: admin@io
    Default password: admin.io
